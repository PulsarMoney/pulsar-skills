# Pulsar Skills

A collection of AI agent skills for building high-quality frontend interfaces, components, and animations.

## Prerequisites
- Node.js

## Installation

```bash
npx skills add PulsarMoney/pulsar-skills
```

## Update skills
```bash
npx skills update
```

## View Skills
```bash
npx skills list
```

## Remove Skills
```bash
npx skills remove <skill-name>
```
